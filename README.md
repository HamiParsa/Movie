## 🎥 About Movie Explorer

**Movie Explorer** is a modern and interactive web app for browsing movies.  
It’s designed to deliver a smooth and visually engaging user experience — from browsing movie posters to watching trailers — all in one place.

Built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**, this project demonstrates the best of modern front-end development with clean design and dynamic interactivity.

---

## 🍿 Features

<ul>
  <li>🏠 <b>Home Page</b> displaying movies in an elegant and responsive grid layout</li>
  <li>🎬 <b>Movie Detail Page</b> with poster, title, description, genre, year, rating, and trailer</li>
  <li>🔍 <b>Search Functionality</b> to quickly find your favorite movie by title</li>
  <li>🧭 <b>Navigation Bar</b> for smooth and simple page transitions</li>
  <li>💫 <b>Framer Motion Animations</b> for dynamic and modern UI transitions</li>
  <li>📱 <b>Fully Responsive</b> design for desktop, tablet, and mobile devices</li>
</ul>

## 💡 Description

The **Movie Explorer** project is a great demonstration of building an engaging, data-driven web app using the modern front-end stack.  
Users can:
- 🎥 Browse movies with stylish animations  
- 🔎 Filter movies using real-time search  
- 🧾 Read detailed descriptions and genres  
- ⭐ View ratings and watch trailers  
- 💻 Enjoy a seamless, responsive, and visually captivating experience  

Perfect for anyone looking to learn and showcase skills in **React**, **Next.js**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.
